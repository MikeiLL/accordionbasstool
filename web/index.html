Exploring the Board

Below you can see a typical accordion bass board. Moving around the board will show the corresponding row and column. Each row is a different type of chord/note. Each column is different note, with adjacent columns being a 5th apart. (You can also select Board Vertical and turn off the highliting in the Options menu.)

Sound

Clicking a button will play a synthesized MIDI sound for that note or chord. Sound can be configured, or turned off in the Sound Menu. The Arpeggiate Chord option will play the notes of a chord in sequence rather than all at once. A different MIDI instrument can be selected for the Bass buttons and the Chord buttons.
(If there is no sound but sound is set to on, please let me know.)

Button Combining

Combining multiple bass and chord, a more advanced technqiue, allows for almost any chord to be played.
Below, you can Click and Shift+Click to select several buttons at the same time (up to 4). You will see a list possible chords (often more than one) that correspond to those buttons.

Redundant Buttons

On the accordion, when certain buttons are pressed, buttons that have the same notes are automatically pressed, that is, they will fall through. This is the intended effect of the Stradella system, buttons that are connected to the same notes have no effect.
To see which buttons are made redundant, the notes by them are already pressed, select Press Redundant Buttons item in the Options.

Chord Picker

Below you can select any chord and see the different ways that it can be played on the accordion bass side. You can choose from an existing list of chords, and also edit the chord by manipulating the steps within that chord. Changing the root will move the chord to that note.

Chord Inversion

Optionally, you can add an inversion to a chord. Specifying an inversion will imply that the root note is played in the bass and that it is the lowest note played. All other inversions of the chord will be filtered out.

Possible Button Combos Table

The table on the right lists the button possibilities for the currently selected chord. The lowest note is specified in parantheses.
Mid C = the lowest C played by a chord button.
Bass C = the lowest C played by a bass button, lower than the Mid C. If an inversion is set, only button combos with the correct root in the bass will be displayed.

Common Progressions & Scales

Below you can see some common progressions that are played on the accordion bass.

Progressions

Provided are two of the simplest rhythms, the Oom-Pah (Polka) and the Waltz. (More Rhythms to be added soon!) The basic pattern alternates between the root and the chord note. Some common enchacements, such as alternating between the 5th and sometimes the 3rd of the chord being played, are also available. One a progression is set, click the Play to simulate the pattern.

Scales

A few common scales have been added as well. The scales only use bass notes and are the basis of playing melody and bass runs on the accordion bass. (More to be added!)

Multiple Seqs

By selecting the Multiple Seqs button, you can build a list of progressions and string them together. Any Progression or Scale can be combined together to form a long progression.

Custom Progression

Here you can enter a completely custom sequence of chords and notes, seperated by a comma. ex. A, B, C, Am, E7

Chord Abbreviations

All Chord abbreviations available in the chord picker can be entered. (Refer to the list of chords in the Chord Picker for full list) (Note: Add List here?)

Listing Notes In A Chord

You can also list notes in a chord entirely by placing it in brackets. Ex. [A C E], E is the same as writing Am, E Any combinations of notes can be included in the brackets. Ex. [A B C D E]

Recent Progression History

All entered progressions are saved (until Clear Recent is clicked) If you've made a mistake, or want to go back to a previous progression, click Show Recent or the drop down on the right to see and select from a list of recent progressions.

Accordion Bass Analyzer

Below you can see the whole analyzer put together. Each of the different modes are available as separate tabs in the analyzer. There is also an additional Options tab for advanced searched settings.

Search Options

In the Options tab, you can alter the search settings used to filter out possible button combos. For example, Max Button Combo Size can be adjusted to only allow combinations with up to 2 buttons, a more reasonable limit.

Finger Search (alpha)

There is also an option to include fingering on all the button combinations. It can be enabled through either the Options tab or menu. Finger searching is not quite working correctly for all combinations, but the eventual goal is to have the finger search be a pivotal aspect of the analyzer.
