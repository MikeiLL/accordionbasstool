Exploring the Board

Below you can see a typical accordion bass board. Moving around the board will show the corresponding row and column. Each row is a different type of chord/note. Each column is different note, with adjacent columns being a 5th apart. (You can also view the board vertically and turn off the highliting in the Options menu.)

Button Combining

Combining multiple bass and chord, a more advanced technqiue, allows for almost any chord to be played.
Below, you can Click and Shift+Click to select several buttons at the same time (up to 4). You will see a list possible chords (often more than one) that correspond to those buttons.

Redundant Buttons

When multiple buttons are pressed, other buttons will automatically fall through on the accordion. This is due to the fact that all the notes in those buttons are played by other notes. This is true for duplicate buttons. To see which buttons are made redundant, select the Press Redundant Buttons item in the Options.
